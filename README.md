# Retrieval-Augmented Generation (RAG) Pipeline for PDF Documents
A Python-based RAG system that processes PDF documents, generates embeddings using OpenAI models, stores vectors in FAISS, and retrieves relevant context through semantic search.

## Project Overview
Built a Retrieval-Augmented Generation (RAG) pipeline using:
- LangChain
- OpenAI Embeddings
- FAISS
- PDF Chunking
